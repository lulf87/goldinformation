 # 设计文档 06：数据源接入说明（V1）
 
 ## 1. 目标
 说明本项目各类数据源的接入方式、用途、依赖与降级策略，方便替换与排查。
 
 ## 2. 数据源清单
 
 ### 2.1 行情价格
 - **来源**：Yahoo Finance（通过 `yfinance`）
 - **用途**：黄金期货价格、美元指数等历史行情
 - **说明**：免费数据，存在一定延迟；本地缓存（Parquet）
 
 ### 2.2 新闻数据
 - **来源**：Finnhub News
 - **用途**：新闻具体内容与情绪分析（规则/LLM）
 - **说明**：免费层可用新闻
 
 ### 2.3 实际利率
 - **来源**：FRED（DGS10 + CPI）
 - **用途**：计算实际利率（名义利率 - 通胀率）
 - **说明**：需要 FRED API Key；失败时回退 Yahoo Finance 或固定值
 
 ## 3. 接入优先级与降级策略
 - **新闻数据**：Finnhub →（失败）模拟新闻并提示
 - **实际利率**：FRED → Yahoo Finance → 固定值
 
 ## 4. 配置项（环境变量）
 - `FINNHUB_API_KEY`：Finnhub 新闻
 - `FRED_API_KEY`：FRED 实际利率
 
 ## 5. 合规与风险提示
 - 优先使用官方或授权 API
 - 避免高频抓取与未经授权的数据来源
 - 若数据源不可用，前端需明确提示数据为模拟或缺失
